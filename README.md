Scheme to facilitate the start of nodeJS projects with express

#### Technologies
- nodeJS
- MongoDB

#### Dependencies
- express: ^4.17.1
- mongoose:^5.10.2
- jest: ^26.4.2
- eslint: ^7.7.0

#### What's in the project:
- Hermod-logger (for Express)
- Integration with CircleCI
- EsLint
- Unit Tests with Jest
